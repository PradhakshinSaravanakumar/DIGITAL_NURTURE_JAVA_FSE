# Cognizant Digital Nurture 5.0 – Java Full Stack Engineer
This repository contains the hands-on exercises, assignments, and practice implementations completed as part of the Cognizant Digital Nurture 5.0 Deep Skilling Program for the Java Full Stack Engineer (Java FSE) track.

- Test Driven Framework
- Logging Framework

### Week 2
- Spring Core and Maven
- Spring Data JPA with Spring Boot
- Hibernate

### Week 3
- Spring REST using Spring Boot 3

### Week 4
- Code Quality and SonarQube
- Microservices with Spring Boot 3 and Spring Cloud

### Week 5
- React

### Week 6
- React
- Git
- CI/CD

### Week 7
- DevOps
- Docker
- Cloud Fundamentals
- Generative AI Fundamentals
